## Key Features

### 1. Infinite Scroll Implementation
- LazyVerticalGrid monitors scroll position
- When user reaches second-to-last item
- Triggers `onLoadMore` event
- ViewModel increases page number
- Loads next page of products
- Appends to existing list

### 2. Search Functionality
- User types in search field
- SearchQueryChanged event updates state
- User presses Enter or search button
- Search event triggers search use case
- Results replace current product list

### 3. Category Filtering
- User selects category
- CategorySelected event triggered
- ViewModel resets to page 1
- Loads products for selected category
- Updates product list

### 4. State Management
- All UI state in single HomeUiState
- Immutable state updates
- StateFlow ensures single source of truth
- UI automatically recomposes on state change

## Technology Choices Rationale

| Technology | Reason |
|-----------|--------|
| **Voyager** | Type-safe, KMP-native navigation with good Koin integration |
| **Koin** | Lightweight DI, good KMP support, easy to set up |
| **Ktor Client** | Official Kotlin HTTP client, excellent KMP support |
| **Kamel** | KMP image loading library specifically for Compose |
| **StateFlow** | Built-in Kotlin reactive stream, perfect for MVVM |
| **Result type** | Kotlin standard library error handling |
